## 一、产品概述

### 1.1 产品定位

一拍即传是Camera360旗下针对**摄影师、摄影工作室**推出的专业照片直播软件，联合MIX滤镜大师深度开发，为摄影师提供**活动现场照片直播、自动修图、即时分享**的一站式解决方案。

产品核心定位：**"能自动修图的照片直播"**，帮助摄影师实现即拍照、即上传、即分享的高效工作流程。

### 1.2 核心价值

| 核心价值 | 说明 |
|---------|------|
| **即时直播** | 兼容主流相机，支持有线和无线FTP传输，实现"拍摄到展示仅需5秒"的极致效率 |
| **无限容量** | 无空间与照片数量限制，可上传任意张照片，自由展示拍摄作品 |
| **AI修图** | 内置第七代智能图像处理引擎，提供130+滤镜、AI美颜、AI瘦脸、AI瘦身功能 |
| **极速分享** | 观众扫码即可浏览和下载照片，支持多渠道社交平台分享 |

### 1.3 目标用户

| 用户类型 | 需求描述 |
|---------|---------|
| **活动摄影师** | 需要快速交付照片、提升工作效率、增加品牌曝光 |
| **摄影工作室** | 需要团队协作、客户管理、服务标准化 |
| **活动主办方** | 需要实时获取活动照片、提升活动传播效果 |
| **活动观众** | 需要即时获取现场照片、方便分享社交 |

## 二、核心功能

### 2.1 照片直播功能

#### 2.1.1 即拍即传
- 支持**300+主流相机型号**
- 兼容**有线OTG**和**无线FTP**传输方式
- 相机拍摄的照片瞬间传输到手机并上传至云端
- 实现真正的**边拍边传**

#### 2.1.2 多种传输模式

| 模式 | 说明 |
|-----|------|
| **即拍即传模式** | 照片拍摄后自动传输上传，适合需要实时展示的场景 |
| **手动选传模式** | 摄影师手动选择满意的照片上传，保证照片质量 |
| **双模式同时使用** | 支持有线和FTP无线传输同时使用，满足不同拍摄需求 |

#### 2.1.3 上传选项

| 选项 | 规格 | 适用场景 |
|-----|------|---------|
| **原图上传** | 最高50MB | 需要最高画质的商业摄影 |
| **高清压缩** | 平衡画质与速度 | 一般活动直播 |
| **标清压缩** | 轻量化文件 | 快速传输、网络不稳定场景 |
| **RAW支持** | 专业格式 | 专业摄影师后期需求 |
| **4K超清** | 高分辨率 | 大屏展示场景 |

---

### 2.2 AI修图功能

#### 2.2.1 滤镜修图
- 联合**MIX滤镜大师**，提供超过**130种滤镜**
- 滤镜类型：自定义调色、通用滤镜、胶片滤镜、黑白滤镜
- **0.3秒光速调色**，有效解决曝光不足、色温不准、颜色不够鲜艳等问题
- 支持保存自定义滤镜，自动应用到新拍摄的照片中

#### 2.2.2 AI人像精修

| 功能 | 描述 |
|-----|------|
| **智能美颜** | 自动识别面部瑕疵并修复，保持肌肤纹理真实自然 |
| **精致瘦脸** | 识别人脸特征点，按比例自然瘦脸，轮廓更精致 |
| **智能瘦身** | 识别手臂、腰部、大腿等部位，自动瘦身和身高调整 |
| **细节美肤** | 修复皮肤瑕疵，保持真实质感 |
| **AI调色** | 自动分析照片内容，智能优化色彩参数 |

#### 2.2.3 批量处理
- 支持**一键美化批量处理**
- 媲美LightRoom的高级快调功能
- 提供**88套各类风格主题套装**
- 支持自定义滤镜和历史滤镜快速通道
